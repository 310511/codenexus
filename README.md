## ğŸ§ª How to Run the Project

### Requirements:
- Node.js v18+ / Python 3.10+  
- MongoDB database  
- Blockchain wallet connections  
- Environment variables (see `.env.example`)  

### Local Setup:
```bash
git clone https://github.com/Ayushdevx/SustainLABs
cd SustainLABs
npm install
cp .env.example .env
```
# edit .env with your values
npm run dev
Backend:
```

```
